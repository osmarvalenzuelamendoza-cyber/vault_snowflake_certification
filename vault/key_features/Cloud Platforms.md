## ❄️ Snowflake

### Arquitectura y Plataformas Cloud

Snowflake corre completamente (3 capas, storage, compute, cloud services) sobre infraestructura cloud (aws, gcp o azure). 
Cada cuenta Snowflake se hostea en **un único cloud provider**, pero una organización puede tener cuentas en distintos providers simultáneamente. 

---
### Data Loading desde cualquier cloud

Se puede cargar y descargar data sin importar el cloud:
- **Internal stages** (Snowflake)
- **Amazon S3**
- **Google Cloud Storage**
- **Microsoft Azure Blob Storage**

- Aplica a **Business Critical Edition o superior**
- Refuerza el compliance en gestión de riesgo y regulación
